# Azure Notes

## What is Azure?
Microsoft Azure ia a cloud computing platform that providesservices like VM's, storage, networking, & security.

## Cloud Models
Public Cloud: Services are delivered over the internet.
Private Cloud: Cloud resources used by one organization
Hybrid Cloud: Mix of on-premises & cloud resources
