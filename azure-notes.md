# Azure Notes

## 1. Introduction
- Azure is a cloud computing platform by Microsoft.
- It provides a variety of services including computing, analytics, storage, and networking.

## 2. Categories of Azure Services
- **Infrastructure as a Service (IaaS)**: Offers virtualized computing resources over the internet.
- **Platform as a Service (PaaS)**: Provides a platform allowing customers to develop, run, and manage applications without the complexity of building and maintaining infrastructure.
- **Software as a Service (SaaS)**: Software distribution model in which applications are hosted by a service provider and made available to customers over the internet.

## 3. Key Services
- **Azure Virtual Machines**: On-demand scalable computing resources.
- **Azure Blob Storage**: RESTful object storage solution for unstructured data.
- **Azure SQL Database**: Managed database service for app development.

## 4. Security and Compliance
- Azure meets a broad set of international and industry-specific compliance standards.
- Security and privacy are integrated into the Azure architecture:
  - **Identity and Access Management**: Control access to resources with Azure Active Directory.
  - **Network Security**: Use firewalls and DDoS protection to safeguard your network. 

## 5. Conclusion
- Azure continues to grow and evolve, providing an array of services.
- It enables businesses to scale efficiently while focusing on core operations.
