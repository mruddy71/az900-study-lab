# Core Azure Services

## Compute
Azure VM's allow user/users to run servers in the cloud

## Networking
Azure Virtual Networks helps connect Azure resources securely.

## Storage
Azure Storage provides cloud-based storage for files, blobs, & other data.

## Databases
Azure offers services like AzureSQL Database for managed database hosting.

## Identity
Azure Active Directory (Microsoft Entra ID) helps manage users, authentication, & access.
