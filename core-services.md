# Core Azure Services

## Overview
Azure provides a comprehensive suite of cloud services designed to support modern application development, data management, and infrastructure needs.

---

## Compute
**Description:** Execute applications and workloads in the cloud with flexible, scalable compute resources.

**Key Services:**
- **Azure Virtual Machines** - Deploy and manage Windows or Linux virtual machines with full control over the operating system and software.
- **Azure App Service** - Build and host web apps, mobile backends, and RESTful APIs.
- **Azure Container Instances** - Run containerized applications without managing servers.
- **Azure Kubernetes Service (AKS)** - Orchestrate containerized applications at scale.

---

## Networking
**Description:** Connect, protect, and optimize your Azure resources and on-premises infrastructure with robust networking capabilities.

**Key Services:**
- **Azure Virtual Networks (VNets)** - Create isolated, secure networks to connect Azure resources.
- **Azure Load Balancer** - Distribute incoming traffic across multiple resources for high availability.
- **Azure Application Gateway** - Layer 7 load balancing with advanced routing and SSL termination.
- **Azure VPN Gateway** - Establish secure, encrypted connections between on-premises and Azure networks.

---

## Storage
**Description:** Store, protect, and retrieve data at scale with multiple storage options for different use cases.

**Key Services:**
- **Azure Blob Storage** - Store unstructured data such as documents, media, and backups.
- **Azure File Shares** - Access files over SMB protocol for hybrid and cloud scenarios.
- **Azure Queue Storage** - Enable reliable asynchronous messaging between applications.
- **Azure Table Storage** - Store structured NoSQL data with flexible schema.

---

## Databases
**Description:** Manage relational and non-relational data with fully managed, enterprise-grade database services.

**Key Services:**
- **Azure SQL Database** - Managed relational database with built-in security, updates, and backups.
- **Azure Database for PostgreSQL/MySQL** - Open-source database services with high availability.
- **Azure Cosmos DB** - Globally distributed, multi-model database for any scale.
- **Azure SQL Managed Instance** - Lift-and-shift SQL Server workloads with minimal changes.

---

## Identity & Access Management
**Description:** Manage users, applications, and secure access to resources across your organization.

**Key Services:**
- **Microsoft Entra ID (Azure AD)** - Manage user identities, authentication, and conditional access.
- **Azure Role-Based Access Control (RBAC)** - Grant granular permissions based on roles and responsibilities.
- **Azure Multi-Factor Authentication (MFA)** - Enhance security with additional authentication factors.
- **Azure Key Vault** - Securely store and manage cryptographic keys and secrets.
